# Public Assets

Place the following files in this directory:

## Required Files:

1. **devin-fox-artist.jpg** - Artist profile image (circular crop recommended, minimum 256x256px)

2. **run-n-tell.mp3** - Audio track 1: "Run N Tell"

3. **luv-somebody.mp3** - Audio track 2: "Luv Somebody"

4. **aint-bad.mp3** - Audio track 3: "Ain't Bad"

## Optional:

- **favicon.ico** - Browser tab icon

All files should be placed directly in the `/public` folder at the root level.
